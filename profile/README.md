# Nife Public

This org hosts open sample and demo repositories from [Nife](https://nife.io) — mostly reference apps used in the [Nife deployment docs](https://docs.nife.io) to show what a one-click deploy looks like for different stacks, plus a few standalone demos.

For the core Nife platform, see [nifetency](https://github.com/nifetency).

## Sample deployment apps
Referenced in the [Nife docs](https://docs.nife.io/Deploy) as one-click deploy examples for different stacks:

- **[react_password_generator](https://github.com/nife-public/react_password_generator)** — Sample React app
- **[rails-example](https://github.com/nife-public/rails-example)** — Sample Ruby on Rails app
- **[ruby-example](https://github.com/nife-public/ruby-example)** — Sample Ruby app
- **[image-app](https://github.com/nife-public/image-app)** — Python-based image processing app
- **[go-todo-list](https://github.com/nife-public/go-todo-list)** — Sample Go web app
- **[docker-spring-boot-java-web-service-example](https://github.com/nife-public/docker-spring-boot-java-web-service-example)** — Sample Dockerized Spring Boot web service
- **[docker-hello-world-spring-boot](https://github.com/nife-public/docker-hello-world-spring-boot)** — Sample Dockerized Spring Boot hello-world app

## Other demos
- **[pingpong](https://github.com/nife-public/pingpong)** — Cognitive assistant that recommends which side to hit a ping-pong ball to
- **[python-builtin](https://github.com/nife-public/python-builtin)** — Simple image processing app
- **[nife-ui-demo](https://github.com/nife-public/nife-ui-demo)** — UI demo of the Nife teamboard
